The Clover Rescue Project

Project

Our project was created in order to help pilots return their flew away drones!

So, how it works?

Having installed the necessary hardware and software on the drone, you can go to our website and be able to monitor the state of the drone and return it home with one click, even if the radio connection is completely lost!
Moreover, in the event of an emergency situation, our software will determine it and offer options for further actions.




Required Hardware

The all you need - Raspberry Pi 3/4, camera and USB WIFI Modem!
